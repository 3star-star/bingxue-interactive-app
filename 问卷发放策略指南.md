| 场景 | 时机 | 预期回收率 | 话术建议 |

| 排队等缆车 | 等待5分钟+ | 70-80% | "您好，排队时帮我们填个3分钟的问卷吗？填完送热饮券" |

## 二、问卷设计优化（提高完成率）

### 2.1 移动端适配检查清单
- [ ] 单选/多选按钮够大（至少44×44px）
- [ ] 表格在手机上能正常显示（建议改成单选题）
- [ ] 进度条显示（让用户知道还剩多少）
- [ ] 支持保存草稿（避免中途退出）

### 2.2 问题顺序优化
**原则**：简单→有趣→核心→敏感

**建议顺序**：
1. 基本信息（1-2题，快速进入状态）
2. 有趣的选择题（如Q8功能打分，有参与感）
3. 核心问题（如Q5复购障碍）
4. 付费意愿（放在后面，避免一开始就劝退）

### 2.3 减少流失的技巧
| 流失点 | 原因 | 解决方案 |
|--------|------|----------|
| 第1题后 | 问题太无聊 | 第1题改成"你最喜欢滑雪的什么？" |
| 表格题 | 太复杂 | 改成单选题 + 进度条 |
| 付费问题 | 太敏感 | 前面加"以下是假设场景"缓冲 |
| 最后1题 | 已经累了 | 最后加"感谢！您的反馈很重要" |

### 4.2 问卷设置技巧
- **必填题**：只设置Q1/Q7/Q8为必填，其他选填（降低放弃率）
- **逻辑跳转**：
  - Q1选A（从未滑过）→ 跳转到Q4
  - Q1选B/C/D → 跳转到Q5
- **防刷设置**：
  - 同一IP只能提交1次
  - 答题时间<60秒的标记为无效
- **数据导出**：选择支持Excel导出的工具

## 七、常见问题FAQ

### Q1：如果回收率太低怎么办？
**A**：
1. 检查问卷是否太长（超过5分钟）
2. 增加礼品吸引力（改成抽奖送雪票）
3. 优化发放话术（强调"帮助改进体验"）
4. 换渠道（线上改线下）

### Q2：如何判断问卷是否有效？
**A**：
- 答题时间<60秒 → 无效
- 所有题都选同一个选项 → 无效
- Q8/Q7全选5分或全选1分 → 无效

### Q3：50份样本够吗？
**A**：
- 做基础分析：够
- 做精细化分层：建议80-100份
- 做统计显著性检验：建议100+份

### Q4：问卷发放需要多少人？
**A**：
- 线上发放：1人即可
- 雪场现场：2-3人（1人负责引导，1人负责记录）
