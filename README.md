# 冰雪互动应用 - Web版本

一个集知识科普、美景体验、游戏挑战于一体的冰雪互动应用。

## 部署到GitHub Pages

要部署这个应用到GitHub Pages，请按照以下步骤操作：

### 1. 创建GitHub仓库
- 在GitHub上创建一个新的公开仓库
- 克隆仓库到本地

### 2. 部署脚本
运行部署脚本：
```bash
./deploy.sh
```

### 3. 访问应用
部署完成后，你的应用将可以通过以下URL访问：
```
https://<your-username>.github.io/<your-repo-name>/
```

## 本地开发

要运行开发服务器：
```bash
npm run dev
```

应用将在 http://localhost:3000 上运行。
