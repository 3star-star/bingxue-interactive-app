# 🚀 GitHub Pages 部署指南

本目录包含需要部署的HTML文件。

---

## 📦 文件清单

- `我的多元宇宙规划 v1.0.html` - 主页面文件

---

## ⚡ 最快部署方式（3分钟）

### 步骤1：在GitHub创建仓库

1. 访问 https://github.com/new
2. 填写信息：
   - **Repository name**: `college-plan`（可自定义）
   - **Description**: 我的大学规划（可选）
   - **Public** ✅（必须选Public才能免费启用Pages）
   - ✅ **Add a README file**（勾选）
3. 点击 **"Create repository"**

---

### 步骤2：上传HTML文件

1. 进入刚创建的仓库
2. 点击 **"Add file" → "Upload files"**
3. 将 `我的多元宇宙规划 v1.0.html` 拖拽到虚线框内
4. 底部输入 "Add planning html file"
5. 点击 **"Commit changes"**

---

### 步骤3：启用GitHub Pages

1. 在仓库页面，点击顶部 **"Settings"**
2. 左侧菜单选择 **"Pages"**
3. 在 **"Build and deployment"** 下：
   - **Source**: 选择 `Deploy from a branch`
   - **Branch**: 选择 `main` → `/root`
4. 点击 **"Save"**

---

### 步骤4：获取你的公开链接

1. 设置页面会自动显示你的站点URL
2. 格式为：`https://你的用户名.github.io/college-plan/`
3. **等待1-2分钟**，然后在新标签页打开这个链接
4. 就能看到你的交互式规划页面了！

---

## 🔄 更新内容

如果以后修改了HTML文件：
1. 回到仓库页面
2. 点击 `我的多元宇宙规划 v1.0.html` 文件
3. 点击铅笔图标 ✏️ 编辑
4. 修改后拉到最下面，输入更新说明
5. 点击 **"Commit changes"**
6. GitHub Pages会自动重新部署（约30秒）

---

## 🎯 注意事项

- ✅ 必须选择 **Public** 仓库，Private仓库需要付费才能用Pages
- ✅ 首次部署需要等待1-2分钟
- ✅ 生成的链接是永久免费的（每月10GB流量限制，个人使用足够）
- ✅ 支持手机、平板、电脑访问
- ✅ 自动HTTPS加密

---

## ❓ 遇到问题？

- **404页面**：等待1-2分钟，GitHub需要时间构建
- **样式错乱**：检查文件是否完整上传（1372行）
- **想换仓库名**：Settings → Repository name 修改后，Pages链接会自动更新
