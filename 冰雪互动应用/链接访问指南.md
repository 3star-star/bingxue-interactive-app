# 冰雪互动应用 - 链接访问指南

## 应用概述

冰雪互动应用是一个集知识科普、美景体验和游戏化学习于一体的创新应用，支持网页版和微信小程序两个版本。

## 访问方式

### 1. 网页版本访问

**直接访问链接：**
```
https://ice-snow-interactive-app.vercel.app
```

**特点：**
- 支持PC和移动设备访问
- 完整的功能体验
- 支持浏览器缓存和书签
- 适合分享和推广

### 2. 微信小程序访问

**二维码访问：**
![小程序二维码](https://placehold.co/200x200/ffffff/000000?text=二维码)

**访问步骤：**
1. 打开微信APP
2. 扫描上方二维码
3. 点击"进入小程序"
4. 开始使用冰雪互动应用

**或通过微信搜索：**
- 搜索"冰雪互动世界"
- 找到对应的小程序

### 3. 本地开发版本

**网页版本：**
```bash
# 进入项目目录
cd 冰雪互动应用/web-version

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问地址：http://localhost:5173
```

**小程序版本：**
```bash
# 进入项目目录
cd 冰雪互动应用/mini-program

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 使用微信开发者工具打开项目
```
