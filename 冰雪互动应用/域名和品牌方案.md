## 二维码生成方案

### 小程序二维码生成

**在线生成工具：**
1. 访问微信官方小程序码生成工具
2. 输入小程序ID：`wx1234567890abcdef`
3. 生成永久二维码

**生成命令（Node.js）：**
```javascript
const qrcode = require('qrcode');

// 生成小程序二维码
qrcode.toFile('mini-program-qr.png', 'https://mini.icesnow.app', {
  color: {
    dark: '#000000',
    light: '#FFFFFF'
  },
  width: 400
}, function(err) {
  if (err) throw err;
  console.log('二维码生成成功！');
});
```

### 网页版本二维码
