# 一键部署到 Vercel

## 步骤1：上传文件到 GitHub

1. 访问 https://github.com/new
2. 仓库名填写：`college-plan`（或其他你喜欢的名字）
3. 选择 **Public**（免费）
4. ✅ 勾选 "Add a README file"
5. 点击 "Create repository"

6. 进入刚创建的仓库，点击 **"Add file" → "Upload files"**
7. 将 `我的多元宇宙规划 v1.0.html` 拖拽到页面
8. 点击 "Commit changes"

---

## 步骤2：部署到 Vercel

1. 访问 https://vercel.com/new
2. 选择 **"Continue with GitHub"** 登录
3. 选择你刚创建的 `college-plan` 仓库
4. 保持默认设置（Vercel会自动识别为静态网站）
5. 点击 **"Deploy"**

等待1-2分钟，Vercel会自动生成一个类似这样的链接：
```
https://college-plan.vercel.app
```

---

## 步骤3：自定义域名（可选）

如果你有自己的域名：
1. 进入项目 Settings → Domains
2. 添加你的域名
3. 按照说明配置DNS解析
